This directory contains screenshots for the proposal deck.

1. **candidate_view.png** - Candidate Assessment screen showing per-skill aptitude breakdown with rule explanations
2. **counterfactual_matrix.png** - Counterfactual Fairness Matrix showing bias analysis across protected attributes  
3. **governance_dashboard.png** - MRM Review Queue showing bias injection and reviewer workflow

## Capture Instructions

To capture these screenshots:

1. Run `streamlit run src/demo/app.py`
2. Navigate to each screen
3. Select appropriate candidate/role combinations
4. Trigger the key interactions (Generate Assessment, Run Analysis, etc.)
5. Capture full browser window showing the complete interface
6. Save as high-resolution PNG files (minimum 1920x1080)
